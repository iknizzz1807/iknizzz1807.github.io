# Thong Nguyen Portfolio

Static portfolio and B2B landing site for my work as a Software Engineer / AI Engineer.

The site focuses on engineering case studies, AI integration projects, backend architecture, realtime systems, and client-facing B2B engagement terms.

Live site: https://ikniz.id.vn/

## Structure

- `index.html` - main B2B portfolio landing page
- `cases/` - project engineering deep-dives
- `videos/` - video showcase library for intros, demos, and architecture walkthroughs
- `assets/diagrams/` - D2 source files and exported SVG diagrams
- `projects-data.js` - case study data rendered on the homepage
